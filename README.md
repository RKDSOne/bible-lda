#### Introduction

Experimenting with LDA on the bible, to see what I could come up with.

#### Getting Started

First run:

```
make build
```

Then run:

```
make nlp
```

You can experiment with k (the number of topics), 
i (the number of iterations), and alpha + beta (tuning parameters).
